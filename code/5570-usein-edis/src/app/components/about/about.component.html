Project Description

This is a movie management system built with Angular and JSON Server. The application allows users to view, create, edit, and delete movies, as well as view actor details and statistics about the movie database.

Features

  • View a list of movies with sorting and filtering capabilities
  • View detailed information about each movie
  • Create, edit, and delete movies
  • View actor details and their filmography
  • View statistics about the movie database

Technologies Used

Angular 19.1.5
JSON Server
Bootstrap 5
TypeScript
HTML/CSS
